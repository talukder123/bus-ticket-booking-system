# 🚌 Bus ticket booking system

A terminal-based Bus Ticket Management System built with **C++**.  
This project simulates a basic bus ticket booking platform with passenger and admin access.

---

# 🚀 Features

## 👤 Passenger Features
- View available buses
- View seat status
- Book tickets
- Cancel tickets
- Search booking by passenger name
- Login system

---

## 🛠️ Admin Features
- Admin login authentication
- View buses
- View seats
- Manage bookings
- Add bus section *(planned for future update)*

---
